üéØ Google Sheet Setup - Step by Step

Follow these steps to connect your Italy itinerary!

1 Create Your Google Sheet

  1. Go to Google Sheets
  2. Click "+ Blank" to create a new spreadsheet
  3. Name it: "Italy Trip 2025 Itinerary"
‚úÖ Pro Tip: You can also use the CSV template file I created! Find italy-itinerary-template.csv in your project folder.

2 Add Column Headers

In Row 1, add these exact column names (case-sensitive!):

name lat lng order dates accommodation activities notes
‚ö†Ô∏è Important: The first 4 columns (name, lat, lng, order) are REQUIRED. The others are optional but recommended.

3 Add Your Italy Destinations

Here's sample data to get you started. Copy and paste into your sheet:

name lat lng order dates accommodation
Rome 41.9028 12.4964 1 Sept 23-26 Hotel TBD
Florence 43.7696 11.2558 2 Sept 26-29 Hotel TBD

üí° Getting Coordinates:
1. Go to Google Maps
2. Right-click on your location
3. Click the coordinates at the top (they copy automatically!)
4. Paste into your sheet - first number = lat, second = lng

4 Publish Your Sheet

  1. Click File ‚Üí Share ‚Üí Publish to web
  2. In the dialog:
    • Under "Link", select your sheet tab (usually "Sheet1")
    • Keep format as "Web page" or select "Comma-separated values (.csv)"
  3. Click "Publish" button
  4. ‚úÖ Make sure "Automatically republish when changes are made" is checked
  5. IMPORTANT: Copy the URL that appears - you'll need it!
‚úÖ Your URL should look like:
https://docs.google.com/spreadsheets/d/1a2b3c4d5e6f7g8h9i0/edit#gid=0

5 Update Your Website Code

  1. Open the file: script-single-sheet.js
  2. Find this section near the top (around line 6):
const GOOGLE_SHEETS_CONFIG = {
  itinerarySheetUrl: 'YOUR_GOOGLE_SHEET_URL',
  useFallbackData: false
};
  1. Replace YOUR_GOOGLE_SHEET_URL with your actual sheet URL
  2. Save the file
üí° Example:
itinerarySheetUrl: 'https://docs.google.com/spreadsheets/d/1a2b3c4d5/edit#gid=0'

6 Activate the New Page

Choose one of these options:

Option A: Rename Files (Recommended)
  1. Rename index.html to index-old.html (backup)
  2. Rename index-new.html to index.html
Option B: Update Existing File
  1. Open your current index.html
  2. Find the line: <script src="script.js"></script>
  3. Change to: <script src="script-single-sheet.js"></script>

7 Test Your Setup!

  1. Open index.html in your web browser
  2. Press F12 to open Developer Tools
  3. Click the Console tab
  4. Look for these messages:
    • "Loading Italy trip data..."
    • "Loaded X destinations from Google Sheets"
    • "All maps loaded successfully!"
  5. Check that both maps display correctly
‚úÖ Success! You should see:
‚Ä¢ Flight arrivals map with 2 colored routes
‚Ä¢ Italy itinerary map with your destinations
‚Ä¢ Click on map markers to see details!
